## Customization

**Change colors:**
Edit `style.css` and modify the color values:
```css
--primary-color: #6366f1;
--background: #0f172a;
```

**Change word list:**
Edit `script.js` and modify the `words` array:
```javascript
const words = ["your", "custom", "words", "here"];
```
